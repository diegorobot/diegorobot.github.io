web scada

Published: Aug 19, 2022 - Updated: Sep 08, 2022

thingsboard 是一款流行的且功能强大的物联网平台，ffos web scada 使用SVG技术在thingsboard的基础开发但在线组态软件，与thingsboard深度整合，作为thingsboard的实体存在，可以绑定资产设备的属性值和遥测值，实时动画显示.

主要特点

  1. 可在系统管理员工端管理每个租户可以使用的组态部件
  2. 可以在系统管理员端新建，导入新的部件，并设置部件的状态变化，可编辑属性
  3. 租户可以自由使用租户管理员分配的部件进行组态
  4. 组态完成后，通过专用仪表板部件调用，在仪表板配置页面显示
